Installing

Installing via composer (Recommended)

                
composer require Hive/config

Installing via Git 


# Clone the repo
cd helloworld.dev
git clone https://github.com/hive/config.git .



// With in your php file, include the package loader
require 'hive/config/include.php';

Overview

The code is split up into the following classes

  1. Library.php : The actual config library, useful for extending functionality.
  2. Object.php : Class for accessing the config object.
  3. Instance.php : Instance of the object class.

Usage 


// With in your php file, include the package loader
require 'hive/config/include.php';

use Hive\config;
